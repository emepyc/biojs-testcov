## Getting Started
Install the module with: `npm install mycovex`

```javascript
var mycovex = require('mycovex');
mycovex.hello("biojs"); // "hello biojs"
```

## Documentation

#### .hello(name)

**Parameter**: `name`
**Type**: `String`
**Example**: `biojs`

The 'hello' method is responsible for showing a name.

How to use this method

```javascript
mycovex.hello('biojs'); // "hello biojs"
```
